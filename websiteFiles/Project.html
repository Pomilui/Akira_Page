Methods

Before we got started with gathering information on the control scenes, we decided to gather a list of all of the characters that appeared in the movie. We had aslo listed the characters in the order of which they first appeared. When we added the script to the code as well, we also made sure to include time stamps of when the characters speak.

After that, we got around to adding the code for the different scenes involving characters that are attempting to assert control over other characters.

We also went and added the different scenes involving people using their powers against other characters as well. This can refer to someone using psychic powers, or someone shooting their gun at other characters.

Then came the time to begin coding the schema for the website, which we created by making five separate coding documents to be linked, and creating a document specifically made for creating the appearance of the website.

We had also created an ODD file to help detail which characters appear in control scenes, along with splitting them up into categories of who uses superpowers and who doesn't.

We also added code detailing the establishment of power scenes in the movie as well. These scenes were meant to show which characters were using powers against each other, with only a small list of people using their powers in the moivie.

The reason we decided to choose this for our project was because we all like the movie Akira, and we felt it important to do a project based on it to celebrate the 30th anniversary of the movie's release.
