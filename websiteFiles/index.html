Akira: Fluctuations of Power and Control

Welcome to the Akira Project which presents and explores the English translation of the Akira anime script. When we began this project, we imagined that the script would be readily available especially for such a popular anime. However, we were shocked to find otherwise. The only available script was a rip from the 2001 DVD release by Pioneer: Akira Remastered. This project is special because

  • We have improved the script by re-watching the movie several times to describe the usage of powers in select scenes.
  • Our version provides the first annotated version of the English translation made from the original Japanese animated movie from 1988.

The project analyzes the theme of control in the popular anime "Akira". Many characters possess, desire, gain, abuse, and lose control throughout the film.
